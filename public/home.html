Get started in 2 minutes

Get your free
Groq API Key

Scryb uses Groq's Whisper API for transcription. It's completely free — no billing, no limits for normal use.

1
Create a free Groq account FREE
Go to console.groq.com and sign up with Google or GitHub. No credit card required — Groq offers a generous free tier for API usage.
2
Open the API Keys page
After logging in, click "API Keys" in the left sidebar, or go directly to console.groq.com/keys.
console.groq.com
├── Playground
├── → API Keys  ← click here
├── Models
└── Usage
3
Create a new API Key
Click the "Create API Key" button. Give it any name (e.g. "scryb"). Copy the key that appears — it starts with gsk_...
+ Create API Key

Name: scryb
Key: gsk_xxxxxxxxxxxxxxxxxxxx ← copy this
4
Paste it in Scryb Settings
Open the app, click the ⚙️ settings icon in the top right, paste your key in the "Groq API Key" field and click Save. You're done — start transcribing!
